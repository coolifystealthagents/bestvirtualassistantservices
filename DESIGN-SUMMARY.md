# Design summary

- Design concept: Editorial magazine
- Layout personality: editorial, staggered columns
- Typography: Playfair Display headings with Source Sans 3 body text
- Color palette: primary #7c2d12, secondary #f59e0b, accent #1d4ed8, background #fff7ed, text #1c1917, muted #78716c
- Shape language: Sharp corners
- Visual motif: oversized serif headlines and article rails
- Unique layout choices: first sections use Split proof ledger, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
